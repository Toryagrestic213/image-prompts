## 二、角色三视图

### 公式

> 基于参考图生成角色设计标准三视图：正视图呈现角色正面全貌，注重身体比例与服饰细节刻画；侧视图展现角色侧面轮廓，强调线条流畅性与立体感塑造；背视图呈现角色背面造型，突出背部装饰与整体协调性。在最左侧添加角色面部正视图的特写，聚焦五官细节与表情刻画，运用细腻光影凸显面部层次感。背景修改成纯白色，整体保持与参考图一致的美术风格和色彩搭配。

## 四、时尚人像摄影提示词

### 公式（一段式连贯描写）

```
摄影风格与器材 → 主体基础特征 → 面部与皮肤细节 → 环境与光影 → 穿搭与配饰 → 情绪与气质 → 画面色调与参数
```

### 各段要素

**1. 摄影风格与器材**
- 风格：日常快照 / 电影感 / 复古胶片 / 时尚杂志 / 街拍纪实
- 器材：Sony A7R5 + 85mm f/1.4 GM / Fujifilm X-T5 + 56mm f/1.2 / Canon EOS R5 + 50mm f/1.2L
- 画质词：8K超清细节 / 35mm胶片质感 / 轻微胶片颗粒感
- 视角：非自拍 / 自拍视角

**2. 主体基础特征**
- 景别：近景特写 / 半身照 / 全身照 / 大特写
- 年龄：具体年龄或范围
- 体型：娇小纤细 / 高挑修长 / 丰满性感 / 运动健美
- 方向：东方女子 / 欧美混血感 / 日系少女 / 韩系氛围美人

**3. 面部与皮肤**
- 肤色：冷白皮 / 暖白皮 / 蜜色肌肤 / 小麦色
- 肤质：白皙透亮 / 水润光滑 / 瓷感哑光
- 触发词：(urfaceid)
- 发型：黑长直 / 齐肩短发 / 精灵切 / 中长卷发 + 状态（自然垂坠/微湿凌乱/风吹飘逸）
- 妆容：裸妆偶像感 / 浓郁烟熏 / 清透水光妆 / 复古红唇

**4. 环境与光影**
- 场景具体化："昏暗的欧式家居风格房间，摆放着台灯等欧式元素"
- 光照：金色侧逆光 / 清晨柔光 / 霓虹灯混合光 / 丁达尔效应 / 百叶窗条纹光影

**5. 穿搭与配饰**
- 服装：材质 + 款式 + 颜色 + 穿着状态（紧致勒肉 / 宽松慵懒 / 一侧滑落露肩）
- 配饰：细窄金丝镜框 / 精致锁骨链 / 层叠耳环 / 贝雷帽

**6. 情绪与气质**
- 表情："随性魅惑有欲感又清纯带着一丝挑逗感"
- 气质：御姐风×纯欲风 / 大小姐高贵优雅 / 清冷禁欲系 / 甜酷辣妹

**7. 色调与参数**
- 色调：低饱和色调 / 高对比暗调 / 胶片暖色调 / 莫兰迪色系
- 比例：ar9:16 / ar1:1 / ar16:9 / ar4:5
