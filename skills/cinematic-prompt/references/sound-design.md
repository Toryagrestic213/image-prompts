## 二、音效层次

### 第1层：环境音
- 自然：雨声、风声、蝉鸣、鸟叫、流水、雷声
- 城市：车流、人声嘈杂、地铁报站、电梯叮咚
- 室内：时钟滴答、空调嗡鸣、炉火噼啪、键盘敲击
- 特殊：太空寂静、水下闷响、洞穴回声

### 第2层：动作音效
- 人物：脚步声、开门声、衣物摩擦、呼吸声
- 物理：金属碰撞、玻璃破碎、木头断裂、布料撕裂
- 格斗：拳击肉声、刀剑碰撞、弓弦声、枪响
- 日常：写字声、翻书声、倒水声、敲门声

### 第3层：情绪音效
- 心跳声（紧张/恐惧）
- 呼吸声（紧张/疲惫）
- 突变音效/Jump Scare（惊吓）
- 静默后的突然声响（悬疑揭示）
- 慢放声音（时间减速感）

### 第4层：特效音效
- 魔法：能量聚集、释放、爆发
- 科技：全息投影、数据流、机械变形
- 国风：剑气、内力、风雷
- 爆炸：冲击波、碎片飞散

## 三、配音/画外音类型

| 类型 | 适用 | 特点 |
|------|------|------|
| 萌系配音 | 漫剧 | 可爱、夸张 |
| 古风配音 | 国风短片 | 古典韵味 |
| 现代自然配音 | 微短剧 | 生活化、真实 |
| 悬疑感配音 | 悬疑/恐怖 | 低沉、压迫 |
| 广告大气配音 | TVC广告 | 简洁有力 |
| 画外音旁白 | 科普/纪录片 | 温柔治愈 / 专业严谨 |
| 简洁slogan | 品牌广告 | 品牌slogan优先 |

## 四、声音设计原则

1. **加入自然环境音和现场音效**
2. **保持对白清晰**，背景音适度
3. **不干扰人物对话**
4. **氛围音效增加场景感**
5. **音乐与画面节奏同步**（卡点）
6. **音量层次分明**：对白 > 音效 > 配乐
