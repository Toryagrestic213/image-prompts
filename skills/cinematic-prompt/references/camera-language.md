# 镜头语言速查表

## 一、运镜方式

| 运镜 | 英文指令 | 效果 | 适用场景 |
|------|---------|------|---------|
| 缓慢推近 | Slow Dolly In | 强调主体，营造紧张/亲密感 | 情感戏、悬疑揭示 |
| 快速猛推 | Fast Zoom In | 冲击力，突出关键帧 | 动作高潮、惊吓 |
| 缓慢拉远 | Slow Dolly Out / Pull Back | 展现场景，营造疏离/孤独感 | 结尾、情绪释放 |
| 快速拉远 | Rapid Zoom Out / Reverse Zoom Out | 震撼揭示全貌 | 史诗场景 |
| 环绕运镜 | Orbital Pan / 360° Orbit | 360度展示主体 | 角色亮相、产品展示 |
| 跟随镜头 | Tracking Shot | 跟随主体移动 | 奔跑、追逐 |
| 左右横移 | Lateral Pan | 展示环境宽度 | 场景建立 |
| 升镜头 | Crane Up / Rising Shot | 从低到高揭示 | 开场、情绪升华 |
| 降镜头 | Crane Down / Descending | 从高到低 | 压迫感 |
| 手持晃动 | Handheld Shake | 纪实感/紧张感 | 战争、恐怖、Vlog |
| 希区柯克变焦 | Dolly Zoom / Vertigo Effect | 背景压缩/拉伸 | 心理扭曲、恐惧 |
| 一镜到底 | One-Take / Long Take | 连续不切换 | 沉浸式、产品广告 |
| 强力变焦 | Macro Fast Zoom | 极近距离急速推进 | 产品细节、微观世界 |
| 俯视平移 | Top View Pan | 俯瞰全景移动 | 空间展示、美食 |
| 低角度追踪 | Low Angle Tracking | 地面视角跟随 | 速度感、力量感 |
| 电影式拉远 | Cinematic Zoom Out | 稳向后拉远 | 品牌落版、结尾定格 |
| 仰拍推进 | Low Angle Push | 仰视角度推进 | 压迫感、英雄感 |
| 高处平稳追踪 | High Angle Tracking | 高处视角平稳跟随 | 全景展示、空间感 |
| 快闪推近 | Flash Zoom In | 镜头突然推进面部 | 冲击感、惊吓 |

### 运镜选择完整列表
- 推近（缓慢推进/快速猛推）
- 拉远（缓慢拉远/快速拉远）
- 环绕（匀速环绕/变速环绕）
- 跟拍（平稳跟拍/晃动跟拍）
- 平移（左右平移/上下平移）
- 摇镜（左右摇/上下摇/弧形摇）
- 变焦（放大/缩小）
- 一镜到底（平稳/轻微晃动）
- 俯拍推进 / 仰拍推进

## 二、景别类型

| 景别 | 描述 | 用途 |
|------|------|------|
| 大远景 | 展示宏大环境全貌 | 开场、史诗感 |
| 远景 | 人物与环境的关系 | 场景建立 |
| 全景 | 人物全身+环境 | 交代人物状态 |
| 中景 | 人物全身/场景局部 | 完整场景/人物群像 |
| 近景 | 人物上半身/产品整体 | 表情+对话 |
| 特写 | 面部/局部细节/产品细节 | 情绪放大、细节展示 |
| 大特写 | 眼睛/嘴唇/极致细节 | 极致情感 |
| 微距 | 极近距离拍摄 | 材质纹理/微观世界 |

### 景别选择
- 特写（面部/局部细节/产品细节）
- 近景（人物上半身/产品整体）
- 中景（人物全身/场景局部）
- 全景（完整场景/人物群像）
- 远景（宏大场景/背景铺垫）
- 俯拍、仰拍、侧拍、斜拍

## 三、转场方式

| 转场 | 效果 | 适用 |
|------|------|------|
| 淡入淡出 | 柔和过渡 | 时间跳跃、回忆 |
| 闪切 | 快速切换制造冲击 | 动作/惊吓 |
| 叠化 | 两画面重叠过渡 | 回忆/梦境 |
| 硬切 | 快速衔接无过渡 | 节奏紧凑、快剪 |
| 模糊过渡 | 模糊→清晰 | 视角转换 |
| 缩放过渡 | 放大/缩小切换 | 空间跳跃 |
| 旋转过渡 | 画面旋转切换 | 眩晕/穿越 |
| 擦除过渡 | 一侧擦除切换 | 对比/平行 |
| 匹配剪辑 | 动作/形状匹配切换 | 创意转场 |

### 镜头切换方式选择
- 淡入淡出、闪切、叠化、硬切（快速衔接）
- 模糊过渡、缩放过渡、旋转过渡、擦除过渡

## 四、特殊镜头

- 双重焦点镜头：前景人物清晰，背景人物也清晰
- 反光面镜头：通过水面倒影拍摄人物
- 裂窗镜头：通过破碎玻璃捕捉人物惊恐表情
- 玻璃门透视镜头：玻璃门映出室内外景象
- 电梯门开合镜头：门开关之间切入不同人物状态
- 墙体移动穿越镜头：墙体间切换两个时间场景
- 时间冻结镜头：人物静止，背景保持运动
- 错位双画面镜头：左右分屏讲述两人不同情绪
- 镜头旋转倒转：画面自上而下翻转形成悬念
- 黑场闪现镜头：黑场中人物瞬间显现增强紧张感
- 隔窗远拍镜头：从窗外远远拍摄室内亲密互动
- 头顶直视镜头：俯拍仰望者，强调其渺小
- 肩膀视角镜头：从人物肩膀后紧随其步伐
- 地面低机位：从草丛中仰拍人物奔跑的脚步
- 超长焦镜头：远距离压缩画面景深拉近感
- 虚实变换镜头：焦点在前景转向后景，逐渐模糊
- 对比切换镜头：先亮光再黑暗，突出情绪落差
- 拉镜特写：从全景瞬间拉近至人物表情
- 缓慢升降镜头：从低处缓缓升起呈现全景
